# berpesan-android
Berpesan Android App - Final Project

Classifying incoming SMS in Bahasa Indonesia into 6 categories:
  1. Ham (legitimate message) / Pesan bersih
  2. Fraud / Penipuan
  3. Promotional / Promosi
  4. SMS Banking
  5. Operator
  6. Unknown / Tidak diketahui (out of first 5 categories)
